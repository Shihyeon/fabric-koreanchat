# 주의!
이 버전은 기존 호환되던 Command Block IDE(fork 버전)와 호환되지 않습니다!


- modMenu 설명 변경
- 인디케이터 위치 수정
- 채팅 제안이나 명령어 제안이 나올 때 인디케이터 위치가 자동으로 변경되어 채팅 제안에 가려지지 않도록 수정
- 서버 추가, 책과 깃팬, 커맨드 블럭 화면 등 여러 화면에 애니메이션 인디케이터 적용
- Windows의 IME를 켜고 끌 수 있도록 수정(기본적으로 Ctrl + I)
- 현재 Screen이 KoreanChatPatch 모드에서 지원하지 않는 경우, 자동으로 Windows의 IME를 활성화 하도록 수정
